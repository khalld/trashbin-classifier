## Setup

Create virtual environment

`python3 -m venv venv`

Activate venv

`source venv/bin/activate`

Install requirements libs

`pip install -r requirements.txt`

Run app

`python3 app.py`

## Utils

Exit from venv

`deactivate`

Tensorboard logs:

`tensorboard --logdir logs`
